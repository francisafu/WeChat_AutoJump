# WeChat_AutoJump

## 运行环境
仅支持安装了.Net4.5及以上版本的Windows计算机运行。  
手机端仅支持安卓系统。

## 使用方法

* Step 1：安卓手机打开「系统――关于手机」并连续点击「版本号」五次打开「开发者选项」。
* Step 2：在「开发者选项」中开启「USB调试」。
* Step 3：连接电脑，在手机上确认当前电脑进行调试。
* Step 4: 下载代码库中的WeChatAutoJump.zip并解压。
* Step 5：双击打开exe文件。如手机设置正确，在软件左下角可见状态「已就绪」。
* Step 6：打开游戏，让手机界面处于即将第一次跳跃的状态。
* Step 7：点击「自动」可自动跳跃。「分析/跳跃」可步进跳跃（快捷键A）。「停止」可结束跳跃。
